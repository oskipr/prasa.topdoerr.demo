# PRASA Dashboard

Dashboard mockup para **PRASA – Plataforma Unificada de Operaciones y Analítica Predictiva**

Demo conceptual de cómo TopDoerr integra sensores, estaciones, producción, distribución y alertas predictivas para la Autoridad de Acueductos y Alcantarillados de Puerto Rico.

## Características

- **KPIs Principales**: Producción, consumo, sensores activos y pérdidas (NRW)
- **Mapa Operacional**: Visualización de plantas, estaciones y tanques
- **Sensores y Telemetría**: Distribución de sensores y estado de telemetría
- **Módulos Predictivos**: Falla de bombas, fugas probables y consumo anómalo
- **Alertas Inteligentes**: Lista de alertas con prioridad
- **Pipeline PRASA**: Diagrama del flujo de datos

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

El dashboard estará disponible en `http://localhost:5173`

## Construcción

```bash
npm run build
```

## Tecnologías

- React 18
- Vite
- Tailwind CSS
- Recharts (para gráficos)

## Notas

Este es un mockup estático sin lógica de backend. Todos los datos son valores de demostración.
